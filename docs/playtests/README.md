# プレイテスト残骸ルール

このディレクトリはプレイテスト時に生成されるアーティファクトの置き場です。

## ルール

1. **スクリーンショット**: `screenshots/YYYYMMDD-HHMMSS-<説明>.png` 形式で保存
2. **テスト結果メモ**: `notes/YYYYMMDD-<要件番号>.md` 形式で保存
3. **このディレクトリは `.gitignore` 対象** — コミットしない
4. 問題を修正したら対応するメモに結果を追記し、スクリーンショットは削除してよい
5. `README.md` 本ファイルのみ git 管理する

## ディレクトリ構成

```
docs/playtests/
  README.md          # このファイル（git管理）
  screenshots/       # スクリーンショット（git除外）
  notes/             # テストメモ（git除外）
```
